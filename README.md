# Стилизация фотографии под старину (Tkinter + OpenCV + ООП)

Файл программы: `sepia_vignette_app.py`

Приложение состоит из двух классов:

- **`Processor`** — загрузка изображения (`cv2.imread`) и его обработка
  (сепия через `cv2.transform`, виньетка через попиксельный расчёт
  расстояния от центра), а также сохранение результата (`cv2.imwrite`).
- **`App(tk.Tk)`** — окно с двумя вкладками (`ttk.Notebook`): на первой —
  загрузка изображения, флажок и ползунок; на второй — стилизованный
  результат и кнопка сохранения.

---

## Задача №1 (GUI)
Разместить в двухвкладочном приложении флажок `Checkbutton` «Включить
сепию» и ползунок регулировки затемнения краёв (радиус виньетки от 50
до 300).

### Решение
Интерфейс построен на `ttk.Notebook` с двумя вкладками: «Настройки» и
«Результат». На вкладке «Настройки» размещены кнопка загрузки
изображения, `Checkbutton` с переменной `BooleanVar` («Включить сепию»)
и `Scale` с диапазоном `from_=50, to=300`, связанный с переменной
`IntVar` (радиус виньетки). На вкладке «Результат» отображается
итоговое изображение и кнопка сохранения.

### Результат работы
При запуске приложения на первой вкладке отображаются элементы
управления: кнопка загрузки файла, флажок сепии и ползунок радиуса
виньетки; на второй — область для вывода обработанного изображения.

## Задача №2 (ООП)
Спроектировать класс. Реализовать умножение каналов матрицы на
коэффициенты сепии с помощью функции `cv2.transform` и ограничением
значений до 255.

### Решение
Создан класс `Processor` с константой `SEPIA_MATRIX` — матрицей
3×3 стандартных коэффициентов сепии для каналов B, G, R. Метод
`apply_sepia` применяет `cv2.transform(img, SEPIA_MATRIX)`, после чего
результат ограничивается функцией `np.clip(..., 0, 255)` и приводится к
типу `uint8`, чтобы не допустить переполнения яркости.

### Результат работы
```python
sepia_img = cv2.transform(img, self.SEPIA_MATRIX)
sepia_img = np.clip(sepia_img, 0, 255).astype(np.uint8)
```
При включённом флажке изображение приобретает характерный
коричневатый оттенок сепии.

## Задача №3 (OpenCV)
Реализовать конвейер: если флажок включён — применить фильтр сепии, а
затем наложить поверх маску виньетки (рассчитать расстояние пикселей
от центра и плавно затемнить края матрицы пропорционально ползунку).

### Решение
Метод `apply_vignette` для каждого пикселя вычисляет расстояние до
центра изображения (`np.indices` + евклидово расстояние), нормирует
его максимальным расстоянием до угла кадра и строит маску затемнения,
плавно убывающую к краям. Сила затемнения зависит от значения ползунка
(`radius / 300`): чем меньше радиус, тем темнее края. Метод `process`
собирает конвейер: сначала (если флажок установлен) применяется
`apply_sepia`, затем к результату всегда применяется `apply_vignette`.

### Результат работы
```python
def process(self, sepia_enabled, radius):
    img = self.original.copy()
    if sepia_enabled:
        img = self.apply_sepia(img)
    img = self.apply_vignette(img, radius)
    return img
```
При перемещении ползунка на значение, близкое к 50, края изображения
затемняются значительно сильнее, чем при значении, близком к 300.

## Задача №4 (Комплекс)
Отобразить стилизованное под старину изображение на второй вкладке.
Добавить метод сохранения результата в формате JPEG/PNG.

### Решение
После нажатия кнопки «Применить эффекты» вызывается
`Processor.process`, результат сохраняется в атрибуте `self.result_image`
и отображается на вкладке «Результат» через `PIL.ImageTk.PhotoImage`
(так как Tkinter не умеет напрямую показывать массивы OpenCV/NumPy).
Кнопка «Сохранить результат» открывает диалог `filedialog.asksaveasfilename`
с фильтрами PNG/JPEG и вызывает метод `Processor.save`, который сохраняет
изображение через `cv2.imwrite`.

### Результат работы
```python
def save(self, img, path):
    cv2.imwrite(path, img)
```
Пользователь выбирает расширение файла (`.png` или `.jpg`) в диалоговом
окне сохранения, после чего стилизованное изображение записывается на
диск, и на экране появляется подтверждение «Изображение сохранено».

## Как запустить
```bash
pip install opencv-python numpy pillow
python sepia_vignette_app.py
```

1. На вкладке «Настройки» нажмите «Загрузить изображение» и выберите
   файл.
2. Отметьте флажок «Включить сепию» (по желанию) и настройте ползунок
   радиуса виньетки.
3. Нажмите «Применить эффекты» — результат откроется на вкладке
   «Результат».
4. Нажмите «Сохранить результат», чтобы записать файл в формате
   PNG или JPEG.
